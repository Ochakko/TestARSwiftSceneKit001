TestARSwiftSceneKit001をオープンソースにします
著作権はおちゃっこLAB
(Site: http://ochakkolab.moo.jp, https://moo-ochakkolab.ssl-lolipop.jp)にあります
(Mail: info@ochakkolab.moo.jp)


iPhone X以降で動くスマホ用アプリです
カメラ映像の上にARとして３Dキャラクターが表示されます
１、２、３、４のボタンを押すと素早くモーションが切り替わります
モーションが終わるとアイドルモーションに素早く戻ります
ビルドはMacのXCodeで行います

TestARSwiftSceneKit001はAppleのSwiftでプログラミングされています
SwiftはオープンソースでライセンスはApache 2.0 Licenseです

実際に動かすにはApple Developerに参加して
証明書をもらいプロジェクトファイル中で証明書を明示しなければ動きません
オープンソースにする際にこの証明書部分を空白にしてありますので
使用者自身が証明書を取得して設定してください
